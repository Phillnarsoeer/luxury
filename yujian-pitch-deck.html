项目概述

基于西南民族大学法学院大创项目《AI法律工具失效模式与规制路径研究》，构建从"咨询"到"行动"的一站式智能法律服务平台。在原有6个功能模块基础上，扩展至15个功能模块，覆盖法律咨询、文书生成、诉讼分析、知识探索、律师匹配等全场景需求。

新增核心功能详解（一）

01

📄 智能文书中心

覆盖起诉状、合同、律师函、答辩状、上诉状、仲裁申请、授权委托书、和解协议、证据清单、保全申请共10种常用法律文书。左侧填表、右侧实时预览，一键生成专业格式文书。内置智能审查引擎，自动标注诉讼时效、管辖法院、证据完整性等关键风险点。

10种模板实时预览风险审查一键生成
02

⚖️ 诉讼策略分析

输入案情摘要后，AI辅助进行争议焦点分析（权重排序+观点阐释）、胜诉率预估（百分比+风险提示）及类案裁判观点汇总。将"案例检索"升级为"策略分析"，更贴合诉讼准备场景。

争议焦点胜诉率预估类案观点
03

📊 法规动态追踪

覆盖劳动法、数据合规、金融监管、环境保护、知识产权、刑事7大领域的新法速递与修订对比。内置法规数据库，一键生成合规影响报告，解决法律信息时效性痛点。

7大领域新法速递合规报告

新增核心功能详解（二）

04

🧠 法律知识图谱

基于Canvas的交互式可视化知识图谱，将法条、案例、主体、法律概念之间的关系通过节点-边网络直观展示。支持拖拽平移、滚轮缩放、悬停详情，将静态的"条文查阅"升级为动态的"知识探索"。

Canvas渲染节点-边网络交互探索
05

👤 当事人画像与风险评估

结合案件类型、争议金额、事发时间，生成综合风险指数评分。自动检测诉讼时效风险（3年普通时效）、证据不足、合同缺失等预警项。双列展示风险预警与有利因素，为用户提供直观决策参考。

风险评分时效预警证据评估
06

🤝 在线调解与协商指引

覆盖劳动、邻里、消费、家庭、债务5类常见纠纷。提供分步骤调解流程指引、协商话术建议、调解协议模板。将服务延伸至诉讼前的替代性纠纷解决（ADR）环节。

5类纠纷步骤指引话术模板

新增核心功能详解（三）

07

🎓 法律实训模拟

交互式剧情模拟——3个递进场景（交通肇事案 → 证据质证 → 量刑辩论），玩家作为辩护律师在每个节点做出选择。三维度评分系统：法律知识、逻辑推理、职业伦理。面向法学生提供低风险实战演练。

3场景决策树三维评分
08

🔍 证据材料分析

支持上传图片、PDF等非结构化数据（演示模式），AI辅助进行关键信息提取。自动生成时间线梳理，标注证据链缺口（严重缺口/待补强），辅助判断证据完整性和可诉性。

时间线证据链缺口分析
09

🧑‍⚖️ 律师智能匹配

内置8位执业律师数据库（覆盖北京、上海、广州、深圳、成都、杭州、武汉、南京），支持案由+地域双维度筛选。展示律师专业领域、执业年限、案件数量、评分，打通"线上咨询"到"线下服务"闭环。

8位律师双维度筛选线上→线下

LocalStorage 持久化 — 答题进度、错题本、暗色模式偏好本地存储。

设计制作过程

从需求到部署的完整工作流

1. 需求分析与功能规划

基于原版"郁见"6模块（首页/AI对话/法考题库/法律文库/案例查找/工具箱），结合法律实务场景，规划9个新增模块——智能文书、诉讼策略、法规追踪、知识图谱、风险评估、调解指引、实训模拟、证据分析、律师匹配。

2. 设计系统锁定

完整保留原版 Art Deco 法律期刊美学：Cormorant Garamond + 思源宋体、深酒红(#721c24) + 金色(#b8942c) + 暖纸底(#faf6f0)。所有新增模块的视觉语言与原有体系无缝融合——极细边框、金色装饰线、径向渐变纸张纹理、菱形色块。

3. 模块实施 — CSS 先行

完整编写 500+ 行 CSS 变量体系，覆盖所有 15 个模块的布局需求。新增"更多"下拉导航，将 15 个标签页分为核心功能、智能分析、实务工具三个类别。

4. 模块实施 — HTML 结构

构建 15 个 tab-content 区块，每个模块独立布局。智能文书采用左右分栏（表单+预览），诉讼策略采用并排面板，知识图谱使用 Canvas 全幅容器。

5. 模块实施 — JavaScript 逻辑

1,600+ 行原生 JavaScript，包含：导航系统、暗色模式、AI对话(DeepSeek API)、文书生成引擎、策略分析算法、法规数据库、Canvas知识图谱渲染、风险评估评分、调解数据驱动、实训模拟决策树、证据时间线分析、律师匹配筛选。

6. 测试验证与部署

浏览器端逐一验证 15 个模块功能正常、零 JS 错误。通过 Git 推送至 GitHub Pages，主站 URL（yujian.html）同步更新为新版，同时保留增强版独立路径（yujian-enhanced.html）。
